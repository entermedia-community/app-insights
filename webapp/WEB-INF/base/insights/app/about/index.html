Why use MITRE Insights?

MITRE Insights uses the power of IBM Watson to provide the most relevant, cross-cutting results about a particular domain area in one place. The Initial Operating Capability (IOC) is focused on providing intelligent insights on Artificial Intelligence topics. Use MITRE Insights for:

  • Discovering the breadth of MITRE knowledge
  • Researching new business development opportunities
  • Streamlining work shaping and project execution
  • Finding domain subject matter experts

What makes MITRE Insights so powerful?

MITRE Insights searches multiple MITRE data and knowledge assets and provides users with a comprehensive snapshot of MITRE’s expertise—relevant MITRE technical products and sponsor deliverable, project pages, subject matter experts, and more. Leveraging this intelligent insight can help users streamline work shaping activities and set new projects up for success.

MITRE Insights is powered by the MITRE Smart Data Layer (SDL), which incorporates Watson Discovery to create an artificial intelligence (AI) pipeline. This AI pipeline applies machine learning and natural language processing to read unstructured content collections, parse documents into passages, and match those passages to relevant entities such as projects, people, or technical capability areas. The SDL automatically extracts keywords, identifies matching concepts, and generates document summaries. These "semantic clues" enable a user to find relevant content without opening and reading documents.

Continuous relevancy training makes the Watson Content Recommender smarter over time. The more you use it and provide feedback, the smarter it becomes. The initial release contains a subset of data sources relating to AI but the machine learning model will be refined to improve the results for additional topic areas in the future.

MITRE Insights vs. Discovery Bar Search

MITRE Insights lets you search in a new way that MITRE’s other search engines cannot do today. A Project Pages discover bar search on "machine learning" will bring you back project pages that mentioned that exact phrase. A MITRE Insights search will bring you items related to machine learning and will provide a cross-cutting array of item types including domain POCs, contracts and Project Work Statements (PWS), MIP research projects, direct projects, products from MPL, capabilities, platforms, and publicly released documents all in one view to give you a comprehensive view of all the knowledge across MITRE.

What's Next for MITRE Insights

MITRE Insights is currently in its early stages. The Initial Operating Capability (IOC) is focused on providing intelligent insights on Artificial Intelligence topics. The machine learning model will be refined to improve the results for additional topic areas in the future. Eventually users will also be able to see visual trends for documents, expertise, contracts, platforms, and projects, which will show the direction of work trends and potential gaps in our knowledge. We will make continual updates to the search, filters, and the user interface improve the experience. Send feedback and suggestions for enhancing the tool to the MITRE Insights team kwwk@groups.mitre.org

The MITRE Insights Architecture

MITRE Insights is built on the MITRE “Know What We Know (KWWK)” architecture. Currently the architecture consists of a Data Layer, an Intelligence Layer, and a Presentation Layer. The Data Layer is where the data sources consist of the Smart Data Layer (SDL) preprocessing pipeline, which ingests data from multiple MITRE data sources and normalizes, extracts, and processes content. This process occurs entirely within the MITRE firewall and prepares the data for ingestion into the Intelligent Layer of the architecture. The Intelligence Layer ingests the processing data into a suite of IBM Watson AI tools. This layer provides the brains that power semantic cluing within MITRE Insights. Via the IBM Cloud, some of the Watson tools utilized include: Cloud Object Storage, Watson Assistant, Watson Studio, Watson Knowledge studio, and Watson Discovery. Once the data has gone through the Watson Processing it is ready to be presented in MITRE Insights, which sits in the Presentation Layer of the KWWK architecture.
